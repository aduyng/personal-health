Fitbit Labs Enrollment Export

Fitbit Labs Enrollment Export contains the aggregated enrollments data collected when interacting with the Fitbit Labs framework.

Files Included:
----------

LabsEnrollmentRecords.csv

Contains all user or system events that contribute to the user's enrollment in a lab.

  lab_id                          - Id of the lab the user is enrolled in
  enrollment_state                - current state of the enrollment (eligible, enrolled, canceled, completed,withdrawn, action_needed)
  enrollment_type                 - type of the enrollment (standard or group)
  external_eligibility_state      - indicates whether additional eligibility requirements are met
  mobile_app_build_variant        - build variant of the app
  last_modified                   - last modified time of enrollment
