User Workouts Data Export

The User Workouts data file includes all of the content you have added to Saved Workouts in the app.

----------

User Workouts.csv

   user_id                                                                 - Unique id for the user
   workout_Name                                                            - Name of the stored workout
   workout_name_localization-id                                            - Localization key of the stored workout name
   workout_type                                                            - Type of the stored workout, e.g. Running

   warm_up_block_name_localization_id                                      - Localization key of the workout warm up block name
   warm_up_block_name                                                      - Name of the workout warm up block
   warm_up_round_name_localization_id                                      - Localization key of the workout warm up round name
   warm_up_round_name                                                      - Name of the workout warm up round
   warm_up_round_cycles                                                    - Number of cycles in loop round
   warm_up_round_duration_micro_sec                                        - Fixed time round duration in micro seconds
   warm_up_round_segment_name_localization_id                              - Localization key of the workout warm up round segment name
   warm_up_round_segment_name                                              - Name of the workout warm up segment
   warm_up_round_segment_type                                              - Type of the workout warm up segment: work or rest
   warm_up_round_segment_goal_type                                         - Type of the workout warm up segment goal, e.g. distance
   warm_up_round_segment_movement_goal_type                                - Type of the movement goal, e.g. distance in mm
   warm_up_round_segment_movement_exact_value                              - Value of the movement goal, e.g. distance in mm
   warm_up_round_segment_movement_range_value_lower_bound                  - For range goal value, lower bound of the movement goal value
   warm_up_round_segment_movement_range_value_upper_bound                  - For range goal value, upper bound of the movement goal value
   warm_up_round_segment_movement_target_type                              - Type of the movement target, e.g. heart rate zone
   warm_up_round_segment_movement_target_exact_value                       - Value of the movement target, e.g. peak
   warm_up_round_segment_movement_target_range_value_lower_bound           - For range target value, lower bound of the movement target value
   warm_up_round_segment_movement_target_range_value_upper_bound           - For range target value, upper bound of the movement target value

   main_block_name_localization_id                                         - Localization key of the workout main block name
   main_block_name                                                         - Name of the workout main block
   main_block_round_name_localization_id                                   - Localization key of the workout main round name
   main_block_round_name                                                   - Name of the workout main round
   main_block_round_cycles                                                 - Number of cycles in loop round
   main_block_round_duration_micro_sec                                     - Fixed time round duration in micro seconds
   main_round_segment_name_localization_id                                 - Localization key of the workout main round segment name
   main_round_segment_name                                                 - Name of the workout main segment
   main_round_segment_type                                                 - Type of the workout main segment: work or rest
   main_round_segment_goal_type                                            - Type of the workout main segment goal, e.g. distance
   main_round_segment_movement_goal_type                                   - Type of the movement goal, e.g. distance in mm
   main_round_segment_movement_exact_value                                 - Value of the movement goal, e.g. distance in mm
   main_round_segment_movement_range_value_lower_bound                     - For range goal value, lower bound of the movement goal value
   main_round_segment_movement_range_value_upper_bound                     - For range goal value, upper bound of the movement goal value
   main_round_segment_movement_target_type                                 - Type of the movement target, e.g. heart rate zone
   main_round_segment_movement_target_exact_value                          - Value of the movement target, e.g. peak
   main_round_segment_movement_target__range_value_lower_bound             - For range target value, lower bound of the movement target value
   main_round_segment_movement_target__range_value_upper_bound             - For range target value, upper bound of the movement target value

   cool_down_block_namelocalization_id                                     - Localization key of the workout cool down block name 
   cool_down_block_name                                                    - Name of the workout cool down block
   cool_down_round_name_localization_id                                    - Localization key of the workout cool down round name
   cool_down_round_name                                                    - Name of the workout cool down round
   cool_down_round_cycles                                                  - Number of cycles in loop round
   cool_down_round_duration_micro_sec                                      - Fixed time round duration in micro seconds
   cool_down_round_segment_name_localization_id                            - Localization key of the workout cool down round segment name
   cool_down_round_segment_name                                            - Name of the workout cool down segment
   cool_down_round_segment_type                                            - Type of the workout cool down segment: work or rest
   cool_down_round_segment_goal_type                                       - Type of the workout cool down segment goal, e.g. distance
   cool_down_round_segment_movement_goal_type                              - Type of the movement goal, e.g. distance in mm
   cool_down_round_segment_movement_goal_exact_value                       - Value of the movement goal, e.g. distance in mm
   cool_down_round_segment_movement_goal_range_value_lower_bound           - For range goal value, lower bound of the movement goal value
   cool_down_round_segment_movement_goal_range_value_upper_bound           - For range goal value, upper bound of the movement goal value
   cool_down_round_segment_movement_target_type                            - Type of the movement target, e.g. heart rate zone
   cool_down_round_segment_movement_target_exact_value                     - Value of the movement target, e.g. peak
   cool_down_round_segment_movement_target_range_value_lower_bound         - For range target value, lower bound of the movement target value
