Time Series Data Export

The Time Series export provides a detailed timeline of your tracked activity.

Files Included:
----------

heart_rate_notification_YYYY-MM-DD.csv                 - Where YYYY-MM-DD is the starting date for the entries in the file.

Each entry has the following values:

    timestamp                         - Date and time at which the entry was logged.
    heart rate notification type                            - Heart rate notification type.
    heart rate threshold beats per minute                            - Threshold beats for minute.
    heart rate trigger value beats per minute                            - Trigger value beats for minute.
